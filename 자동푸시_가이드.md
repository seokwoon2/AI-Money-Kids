# 🚀 자동 푸시 가이드

## 방법 1: 배치 파일 사용 (가장 쉬움!)

### 사용법:
1. 프로젝트 폴더에서 `자동푸시.bat` 파일 더블클릭
2. 커밋 메시지 입력 (또는 Enter로 기본 메시지 사용)
3. 자동으로 푸시 완료!

### 빠른 푸시:
- `빠른푸시.bat` 더블클릭
- 메시지 입력 없이 바로 푸시

---

## 방법 2: VS Code에서 자동 저장 후 푸시

### 설정 방법:

1. **VS Code 설정 열기**
   - `Ctrl + ,` (설정)
   - 또는 File → Preferences → Settings

2. **자동 저장 설정**
   - 검색창에 "auto save" 입력
   - "Files: Auto Save" → "afterDelay" 선택

3. **Git 자동 푸시 확장 설치**
   - 확장 프로그램에서 "Git Auto Push" 검색
   - 설치

---

## 방법 3: VS Code 작업 자동화

### 설정 파일 생성:

`.vscode/tasks.json` 파일 생성:

```json
{
    "version": "2.0.0",
    "tasks": [
        {
            "label": "Git Push",
            "type": "shell",
            "command": "git push",
            "group": {
                "kind": "build",
                "isDefault": true
            },
            "presentation": {
                "reveal": "always"
            }
        }
    ]
}
```

### 사용법:
- `Ctrl + Shift + P`
- "Tasks: Run Task" 입력
- "Git Push" 선택

---

## 방법 4: Git Hooks 사용 (고급)

### 자동 푸시 Hook 설정:

`.git/hooks/post-commit` 파일 생성:

```bash
#!/bin/sh
git push
```

Windows에서는 `.git/hooks/post-commit.bat`:

```batch
@echo off
git push
```

**주의:** 이 방법은 커밋할 때마다 자동 푸시됩니다.

## 문제 해결

1. 인터넷 연결 확인
2. GitHub 로그인 확인
3. VS Code에서 수동 푸시 시도

1. GitHub Personal Access Token 생성
2. Git Credential Manager 사용
