# 🎯 VS Code에서 푸시하기 (초보자용)

## 화면 구성 이해하기

VS Code 화면을 보면:

```
┌─────────────────────────────────────────┐
│  [소스제어] [검색] [확장] ...           │ ← 왼쪽 사이드바
│                                         │
│  ┌─────────────────────────────────┐   │
│  │  파일 편집 영역                  │   │
│  │                                 │   │
│  └─────────────────────────────────┘   │
│                                         │
│  [main] [Python] [UTF-8] ...          │ ← 하단 상태바
└─────────────────────────────────────────┘
```

---

## 방법 1: 소스 제어 패널 사용 (가장 확실함!)

### 1단계: 왼쪽 사이드바에서 소스 제어 아이콘 찾기

**위치:**
- VS Code **왼쪽 가장자리**에 세로로 나열된 아이콘들
- 보통 5-6개의 아이콘이 있음

**찾는 방법:**
- **분기 모양 아이콘** 찾기 (Y자 모양 또는 나뭇가지 모양)
- 또는 **숫자가 표시된 아이콘** (예: "4" 또는 "M" 표시)
- 마우스를 올리면 "Source Control" 또는 "소스 제어"라고 표시됨

**클릭!**

### 2단계: 소스 제어 패널이 열림

열리면 보이는 것들:
- 위쪽: "SOURCE CONTROL" 또는 "소스 제어" 제목
- 중간: "Changes" 섹션 (변경된 파일 목록)
- 위쪽: "Message" 입력란 (여기에 커밋 메시지 입력)

### 3단계: 커밋 메시지 입력

**위치:**
- 소스 제어 패널 **가장 위쪽**
- "Message" 또는 "커밋 메시지" 입력란

**입력:**
```
Update: Gemini API and Streamlit Cloud secrets support
```

### 4단계: 커밋하기

**위치:**
- "Message" 입력란 **오른쪽**에 있는 아이콘들
- **"✓" (체크 표시)** 아이콘 찾기
- 또는 **"Commit"** 버튼

**클릭!**

### 5단계: 푸시하기

**위치:**
- 소스 제어 패널 **가장 위쪽**
- "..." (점 3개) 아이콘 찾기
- "Message" 입력란 **오른쪽**에 있음

**클릭 → 메뉴가 열림 → "Push" 선택**

---

## 방법 2: 하단 상태바 사용 (더 빠름!)

### 1단계: VS Code 하단 확인

**위치:**
- VS Code **가장 아래쪽** (파란색 또는 회색 바)
- 왼쪽에 여러 정보가 표시됨

**찾는 것:**
- **"main"** 또는 **"master"** 텍스트 찾기
- 브랜치 이름이 표시됨

### 2단계: "main" 클릭

**클릭!**

### 3단계: 메뉴에서 "Push" 선택

**메뉴가 열림:**
- "Create Branch..."
- "Rename Branch..."
- **"Push"** ← 이걸 선택!
- "Pull"
- 등등...

**"Push" 클릭!**

---

## 방법 3: 명령 팔레트 사용

### 1단계: 명령 팔레트 열기

**키보드:**
- `Ctrl + Shift + P` 동시에 누르기
- 또는 `F1` 키

**화면 상단 중앙에 검색창이 나타남**

### 2단계: "Git Push" 입력

**입력:**
```
Git Push
```

### 3단계: Enter 키 누르기

**자동으로 푸시됨!**

---

## 💡 팁: 아이콘 위치 기억하기

### 소스 제어 아이콘:
- 왼쪽 사이드바
- 분기 모양 또는 숫자 표시

### 커밋 버튼:
- 소스 제어 패널 위쪽
- "✓" 체크 표시

### 푸시 버튼:
- 소스 제어 패널 위쪽
- "..." 점 3개 메뉴

### 브랜치 이름:
- 하단 상태바 왼쪽
- "main" 또는 "master"

---

## 🆘 찾을 수 없을 때

### 소스 제어 아이콘이 안 보일 때:
- `Ctrl + Shift + G` 키 누르기
- 소스 제어 패널이 열림

### 하단 상태바가 안 보일 때:
- `View` 메뉴 → `Status Bar` 체크 확인

### 명령 팔레트가 안 열릴 때:
- `Ctrl + Shift + P` 다시 누르기
- 또는 `F1` 키

---
