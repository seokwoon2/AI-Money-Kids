## 배포 단계 (5분이면 완료!)

### 1단계: 코드를 GitHub에 푸시

터미널에서 실행:

```powershell
# 변경사항 추가
git add .

# 커밋
git commit -m "Update: Streamlit Cloud secrets support"

# GitHub에 푸시
git push
```

**오류가 나면:**
- GitHub 로그인이 필요할 수 있습니다
- 브라우저에서 GitHub 로그인 창이 열리면 로그인하세요

---

### 2단계: Streamlit Cloud 접속

브라우저에서:
```
https://share.streamlit.io
```

또는 터미널에서:
```powershell
start https://share.streamlit.io
```

---

### 3단계: GitHub로 로그인

1. **"Sign in"** 또는 **"Continue with GitHub"** 클릭
2. GitHub 계정으로 로그인
3. 권한 허용 클릭

---

### 4단계: 새 앱 배포

#### 방법 1: 메인 화면에서
1. **"New app"** 버튼 클릭
   - 또는 오른쪽 상단의 **"+"** 아이콘 클릭

#### 방법 2: 직접 URL
```
https://share.streamlit.io/deploy
```

---

### 5단계: 배포 설정 입력

다음 정보를 입력:

**Repository (저장소):**
```
seokwoon2/AI-Money-Kids
```
또는 드롭다운에서 선택

**Branch (브랜치):**
```
main
```
(기본값)

**Main file path (메인 파일):**
```
app.py
```

**App URL (앱 URL):**
- 자동 생성됨
- 예: `ai-money-kids.streamlit.app`

---

### 6단계: 배포 시작

**"Deploy!"** 버튼 클릭

---

### 7단계: 배포 진행 확인

- "Building..." 메시지 표시
- 2-5분 정도 소요
- 진행 상황이 화면에 표시됨

---

### 8단계: 배포 완료 확인

- "Your app is live!" 메시지 확인
- 앱 URL 확인:
  ```
  https://ai-money-kids.streamlit.app
  ```
  (실제 URL은 다를 수 있습니다)

---

### 9단계: API 키 설정 (중요!)

배포는 완료되었지만, API 키를 설정해야 앱이 작동합니다.

#### 9-1. 앱 설정 열기
1. 앱 페이지에서 **⚙️ (톱니바퀴) 아이콘** 클릭
2. 또는 **"Manage app"** → **"Settings"** 클릭

#### 9-2. Secrets 탭 선택
- 왼쪽 메뉴에서 **"Secrets"** 클릭

#### 9-3. API 키 입력
다음 내용을 입력:

```toml
[secrets]
GOOGLE_API_KEY = "AIzaSyAFTERBd0vyHCKWckgeXbubZ6tm9sjr-i8"
```

**⚠️ 중요:**
- `[secrets]`는 반드시 첫 줄에
- `GOOGLE_API_KEY`는 대문자
- 등호 양쪽에 공백
- API 키는 따옴표로 감싸기

#### 9-4. 저장
- **"Save"** 버튼 클릭
- "Saved!" 메시지 확인
- 앱이 자동으로 재시작됨

---

## 🔧 문제 해결

### "New app" 버튼이 안 보일 때
- 오른쪽 상단의 **"+"** 아이콘 클릭
- 또는 직접 URL: `https://share.streamlit.io/deploy`

### Repository가 안 보일 때
- GitHub 저장소가 Public인지 확인
- 저장소 이름이 정확한지 확인: `seokwoon2/AI-Money-Kids`

### 배포 실패 시
1. **Logs 확인**: 앱 페이지 → "Logs" 탭
2. **requirements.txt 확인**: 모든 패키지가 올바른지 확인
3. **app.py 확인**: 파일이 프로젝트 루트에 있는지 확인

### API 키 오류 시
1. Secrets 형식 확인
2. 앱 재시작: Settings → "Reboot app"
3. 로그 확인: Settings → "Logs"
